<b>1.) Home Page</b>
<p>Halaman awal menamppilkan keseluruhan data.</p>
<p><img src="resources/images/1_home_page.JPG" width="800"></p>

<b>2.) Halaman Tambah Data</b>
<p>Melakukan input data terdiri dari Nama Lengkap, Email, No. Hp dan Alamat. Button Simpan untuk menyimpang data sedangkan button Kembali untuk kembali ke halaman home.</p>
<p><img src="resources/images/2_tambah_data.JPG" width="800"></p>
<p>Muncul alert yang berarti field Nama Lengkap, Email dan No. Hp wajib di isi.</p>
<p><img src="resources/images/3_tambah_field.JPG" width="800"></p>
<p>Field saat terisi inputan kemudian klik button Simpan.</p>
<p><img src="resources/images/4_tambah_terisi.JPG" width="800"></p>
<p>Secara bersamaan muncul alert untuk menandakan bahwa data berhasil disimpan.</p>
<p><img src="resources/images/5_tambah_berhasil.JPG" width="800"></p>

<b>3.) Halaman Edit Data</b>
<p>Tampilan halaman ketika setelah melakukan klik pada button Edit.</p>
<p><img src="resources/images/6_edit_data.JPG" width="800"></p>
<p>Muncul alert yang berarti field Nama Lengkap, Email dan No. Hp wajib di edit / tidak boleh kosong.</p>
<p><img src="resources/images/7_edit_field.JPG" width="800"></p>
<p>Field saat terisi inputan/sudah di edit kemudian klik button Simpan.</p>
<p><img src="resources/images/8_edit_terisi.JPG" width="800"></p>
<p>Muncul alert untuk menandakan bahwa data berhasil di edit.</p>
<p><img src="resources/images/9_edit_terupdate.JPG" width="800"></p>

<p>Klik button Cetak PDF kemudian akan men-download file laporan dalam bentuk pdf.</p>
<p><img src="resources/images/10_pdf.JPG" width="800"></p>

<b>5.) Halaman Delete Data</b>
<p>Ketika melakukan klik button Delete data akan terhapus dan secara muncul alert untuk menandakan data berhasil dihapus.</p>
